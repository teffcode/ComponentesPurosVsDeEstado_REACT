📦  COMPONENTES PUROS VS COMPONENTES DE ESTADO 📦  

  		También conocidos como: 

🎨  COMPONENTES DE UI Y COMPONENTES CONTENEDORES 🐳  

   - Es una forma de manejar nuestros componentes
   - Propuesta de que dividamos los componentes en 2 tipos



🐰⭐  ️COMPONENTES PUROS:
	
	- También simplemente COMPONENTES se crean usando FUNCIONES
	- Se crean para formar parte de la UI

🐰 ⭐ ️COMPONENTES CONTENEDORES:

	- Se crean a partir de las CLASES 
	- Ya que un CONTENEDOR tiene LÓGICA pero no debería tener UI



💜  Se separa la UI de la LÓGICA.
💜  Para facilitarnos a nosotros mantener la app.
💜  Así, la función HOLA la puedo usar en otra parte de la app.
💜  Esa es la principal ventaja de trabajar con componentes.
